# Análise de Dados Públicos de Segurança

Projeto desenvolvido para a Atividade Extensionista II do CST em Ciência de Dados.

O objetivo é analisar dados relacionados a ocorrências urbanas nos bairros Setor Bueno e Setor Marista, em Goiânia – GO.

Tecnologias utilizadas:

- Python
- Pandas
- Matplotlib
- Google Sheets
